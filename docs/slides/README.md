# Presenter deck

A short, self-contained slide deck for the workshop presenter.

Open `presenter-deck.html` in any browser. No build step, no internet, no dependencies.

## Controls

- Right arrow, space, or click: next slide
- Left arrow: previous slide
- Home / End: first / last slide
- `N`: toggle presenter notes (shown at the bottom of each slide)

## Contents

13 slides covering the exploit-then-defend flow: the app and its two-app architecture, the threat model (alice the attendee, bob the organizer), why agent security differs from app security, and the four modules (indirect prompt injection, broken authorization via MCP token propagation, excessive agency, sensitive disclosure), ending on the linchpin idea: carry the user's identity to where the decision is made.
